# Music Player Bot For Telegram Group Voice Chat ...



### Commands 

- `/play <song name>` - play song you requested 
- `/pause` - pause song play
- `/resume` - resume song play
- `/skip` - play next song
- `/end` - stop music play
- `/reload` - Refresh admin list
